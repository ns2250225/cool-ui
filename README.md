# Cool-UI
🎨  一个炫酷的UI框架，基于Vue2

# 在线DEMO
- http://www.ns2250225.com/

# 本地组件演示
- npm install
- npm run docs:dev

# 安装使用
- npm install my-cool-ui --save
- 引入
```
# 引入css
import 'my-cool-ui/package/coolui.min.css'  

# 引入coolui.min.js
import coolui from 'my-cool-ui/package/coolui.min.js'  

# 注入到vue
Vue.use(coolui)
```
